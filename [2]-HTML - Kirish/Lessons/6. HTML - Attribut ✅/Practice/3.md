*Paragrafni* qizil rangga bo'yang:

~~~html
<p>
    Qizil rangdagi matin    
</p>
~~~

:::main{#warn}
`<style>...</style>` tegining attribut analogini qo'llang
:::

---ECHIM
Qizil ranggi ingliz tilida *red*, rang berish uchun esa *color* CSS qoyidasi ishlatiladi:

~~~html
<p style="color: red;">
    Qizil rangdagi matin
</p>
~~~
